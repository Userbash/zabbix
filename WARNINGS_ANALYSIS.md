- Поведение образа не меняется

**Как исправить:**
```bash

# Опция 2: Принять это (рекомендуется)
# Это ожидаемое предупреждение при использовании Podman с OCI

# Опция 3: Удалить SHELL директиву из Dockerfile (не рекомендуется)
# Параметры -o pipefail необходимы для надежности
```

## 📝 ПРИЛОЖЕНИЕ: ПОЛНАЯ КЛАССИФИКАЦИЯ

### Что это SHELL warning?

**Dockerfile:**
```dockerfile
SHELL ["/bin/sh", "-o", "pipefail", "-c"]
```

**Проблема в Podman:**
- OCI spec не поддерживает SHELL di директиву
- Podman при OCI format игнорирует это

**Почему это в Dockerfile:**
- Docker поддерживает это
- Параметр `pipefail` гарантирует, что ошибки в pipe обнаруживаются
- Необходимо для надежности

**Решение:**
```bash
# Это не проблема - работает везде правильно!
```
